Funcionament¶

class Usuari.UsuariBank(bankAdvanced, name, passw)[source]¶

Aquesta classe gestiona un usuari d’un banc. Té funcions per autentificar, printejar la compta, depositar,retirar i aplicar el procés mensual.

Per què un usuari entri s’autentifica amb un màxim de 3 intents i podrà depositar o retirar segons l’estat del seu compte.

authentificate()[source]¶

En aquesta funció t’autentifiques escrivint el nom i la contrasenya en 3 intents com a límit. :return: retorna true si accedeixes a la compta i false si no.

comptaBancaria()[source]¶

Printeja les dades de la compta.

deposit()[source]¶

Funció per ingressar diners al compte.

monthlyProcess()[source]¶

Funció que aplica el procés mensual al compte.

withdraw()[source]¶

Funció per retirar diners. Si la teva compta està inactiva no pots retirar diners.

Usuari.checkIfFloat(number)[source]¶

Mira si un numero és float o no :param number: número donat :return: retorna true si és float i false si no
